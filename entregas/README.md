# 📤 Carpeta de Entregas — Estudiantes

Esta carpeta contiene las entregas semanales de todos los estudiantes.

## Estructura de entrega

```
entregas/
├── SEMANA-01/
│   ├── u-20221001/           ← Código del estudiante
│   │   ├── GT01_completado.md
│   │   ├── LAB01_completado.md
│   │   └── capturas/
│   │       └── unity_escena.png
│   └── u-20221002/
│       └── ...
├── SEMANA-02/
└── ...
```

## Instrucciones

1. Navegar a `SEMANA-XX/`
2. Crear carpeta con tu código: `u-TUCODIGO/`
3. Copiar plantilla de guía de trabajo y completarla
4. Copiar plantilla de laboratorio y completarla
5. Agregar capturas en subcarpeta `capturas/`
6. Hacer commit y push → abrir PR

**No modificar** carpetas de otros estudiantes.
