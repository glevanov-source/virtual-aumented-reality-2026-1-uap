## 🚀 Inicio Rápido para Estudiantes

```bash
# 1. Hacer FORK de este repositorio (botón Fork en GitHub)
# 2. Clonar tu fork:
git clone https://github.com/TU-USUARIO/rva-2026-1-uap.git
cd rva-2026-1-uap

# 3. Crear tu rama de trabajo:
git checkout -b u-TU-CODIGO-semana-01

# 4. Trabajar y guardar:
git add .
git commit -m "GT01 completado - TU NOMBRE"
git push origin u-TU-CODIGO-semana-01

# 5. Abrir Pull Request en GitHub → el docente revisará
```

👉 **Lee la guía completa:** [GUIA-ESTUDIANTE.md](GUIA-ESTUDIANTE.md)
